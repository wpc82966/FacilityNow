☁️ Firebase Sync Setup

Connect to Firebase to sync changes across your phone, laptop, and with your team in real time. Free setup takes ~5 minutes.

📋 How to set up (tap to expand)
  1. Go to console.firebase.google.com and sign in with a Google account.
  2. Click "Add project" → name it cfrc-facilities → skip Google Analytics → Create project.
  3. In the left sidebar click Build → Firestore Database → Create database → choose Start in test mode → select us-east1 → Enable.
  4. In the left sidebar click Project Settings (gear icon) → scroll to "Your apps" → click </> (Web) → register app as cfrc-facilities.
  5. Copy the firebaseConfig object values into the fields below and tap Save.
